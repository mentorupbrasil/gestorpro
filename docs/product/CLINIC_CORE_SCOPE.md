# Escopo Clínica Core

Inclui plataforma segura; empresas e trabalhadores; PCMSO/protocolos; encaminhamento/agenda; check-in, etapas, filas e painel; triagem/consulta; exames essenciais; ASO/documentos; financeiro básico; portal empresarial; auditoria, observabilidade, backup e contingência.

Cada fatia só é pronta quando funciona de ponta a ponta com autorização por recurso, RLS, integridade, transação, auditoria, estados de UI e testes negativos. Checkpoint de schema ou tela não equivale a entrega.
